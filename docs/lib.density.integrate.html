lib.density.integrate.importanceSampling module¶

class lib.density.integrate.importanceSampling.ImportanceSampleIntegrateUniform(f, p, ranges=None)[source]¶

Bases: object

Importance Sampling is the idea that we want to solve the integral of the form:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]

It is entirely possible that one samples from the distribution \(p(\mathbf w)\). Under this formulation, the equation for importance sampling (in the limit of infinite samples \(N\)) would be given by the following:

\[\mathbb E(f) = \int_{-\infty}^{\infty} f(\mathbf w) p(\mathbf w) d \mathbf w\]
\[\mathbb E(f) = \frac 1 N \sum_{i=1}^N {f(w_i)}, w_i \sim p(\mathbf w)\]

initialize the Importance Sampler.

Parameters

  • f (function) – This takes a numpy nd-array of dimensions \((N,d)\) representing a set of \(N\) vectors of dimension \(d\) and return \(N\) values that needs to be integrated over the \(d\) dimensional vector space.

  • p (function) – This takes a numpy nd-array of dimensions \((N,d)\) representing a set of \(N\) vectors of dimension \(d\) and return \(N\) values that represents the probability density function over the \(d\) dimensional space.

  • ranges (list of lists, optional) – This is a list of \(d\) ranges, one for each dimension over which the \(d\) dimensional vector space is going to be uniformly sampled, by default None

integrate(N, ranges=None)[source]¶

Integrate the function f over the \(d\) dimensional space.

The integration is performed as a sum with \(N\) points sampled throough rejection sampling with the probability density p.

Parameters

  • N (int) – Number of points to use as samples for the integration

  • ranges (list of lists, optional) – This is a list of \(d\) ranges, one for each dimension over which the \(d\) dimensional vector space is going to be uniformly sampled, by default None

Returns

The result of the integration.

float
